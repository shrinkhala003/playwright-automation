# UI Automation - Playwright

This repository contains Playwright automation projects.

## Projects:

### 1. cm-basecamp
- Automation tests for Basecamp application

### 2. pom-playwright
- Playwright framework using Page Object Model (POM)

## Tools Used:
- Playwright
- TypeScript

## How to use:
- Navigate to respective project folder
- Install dependencies using npm install
- Run tests using npx playwright test
